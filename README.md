# Mithila Madhara Portfolio

A modern, responsive portfolio website for Mithila Madhara, a UX Designer with background in Information Technology and Interactive Media.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Navigation**: Sticky navbar with smooth scrolling to different sections
- **About Me**: Profile information, academic background, and skills
- **Work Portfolio**: Showcase of design projects with images
- **Contact**: Email and social media links
- **Modern UI**: Clean, minimalist design with dark header and light content areas

## Structure

- `index.html` - Main HTML file with all page sections
- `styles.css` - Complete styling with responsive design
- `script.js` - JavaScript for smooth scrolling and interactive features
- `profile.jpg` - Profile image (add your own)
- `portfolio-1.jpg`, `portfolio-2.jpg`, `portfolio-3.jpg` - Portfolio project images

## Getting Started

1. Replace the placeholder images with your actual images:
   - `profile.jpg` - Your profile photo
   - `portfolio-1.jpg`, `portfolio-2.jpg`, `portfolio-3.jpg` - Your portfolio project images

2. Update the contact information in the Contact section with your actual email and LinkedIn profile

3. Open `index.html` in your web browser

## Customization

- Update the text content in `index.html` to match your information
- Modify colors in `styles.css` by changing the color values
- Add or remove portfolio items by duplicating/removing `.portfolio-card` elements
- Update contact links with your actual email and social media profiles

## Browser Support

- Chrome/Edge (Latest)
- Firefox (Latest)
- Safari (Latest)
